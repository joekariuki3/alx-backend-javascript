## ES6 Promises

Exercises focused on working with Promises: creating, chaining, parallelization with `Promise.all`, error handling with `try/catch`, and async/await.

### Prerequisites

- Node.js 14+
- npm

### Install

```
npm install
```

### Run tests and lint

```
npm test           # run Jest test suite
npm run full-test  # ESLint + Jest
```
